Command Line Options:

Service Commander.exe Load=Filename.ini     

loads specified configuration to the computer 
this is a silent option though Service Cmmander will still pop up 
a results window and after 30 seconds it will close it automatically 
UNLESS you right click on the close button

Service Commander.exe Save=Filename.ini 

Saves the current configuration to the specified file

Note:
The load file must be in the configurations folder

You must have a configurations folder for the save or load function to work

ALSO make sure have the configurations folder with the program executable
in order to use the provided configurations
